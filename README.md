## рҹ“– йЎ№зӣ®д»Ӣз»Қ

**my-rpc-framework** жҳҜдёҖж¬ҫд»Һйӣ¶ејҖе§ӢжүӢеҶҷзҡ„иҪ»йҮҸзә§еҲҶеёғејҸRPCжЎҶжһ¶пјҢеҸӮиҖғDubboзҡ„и®ҫи®ЎжҖқжғіпјҢж—ЁеңЁи§ЈеҶіеҲҶеёғејҸжңҚеҠЎд№Ӣй—ҙзҡ„иҝңзЁӢйҖҡдҝЎй—®йўҳгҖӮжЎҶжһ¶ж ёеҝғд»Јз ҒзәҰ3000иЎҢпјҢе®һзҺ°дәҶжңҚеҠЎжіЁеҶҢеҸ‘зҺ°гҖҒзҪ‘з»ңйҖҡдҝЎгҖҒеәҸеҲ—еҢ–гҖҒиҙҹиҪҪеқҮиЎЎзӯүж ёеҝғеҠҹиғҪпјҢе№¶жҸҗдҫӣдәҶSpring Boot Starterж–№дҫҝеҝ«йҖҹжҺҘе…ҘгҖӮ

### дёәд»Җд№ҲеҶҷиҝҷдёӘйЎ№зӣ®пјҹ

- дёәеҗҺз»ӯеӯҰд№ DubboгҖҒgRPCзӯүе·Ҙдёҡзә§жЎҶжһ¶жү“дёӢеҹәзЎҖ

### 3. е®һзҺ°жңҚеҠЎжҸҗдҫӣиҖ…

```java
// rpc-providerжЁЎеқ—
@Slf4j
public class HelloServiceImpl implements HelloService {
    @Override
    public String sayHello(String name) {
        return "Hello, " + name;
    }
    
    @Override
    public String sayHello(String name, int age) {
        return String.format("Hello, %s! You are %d years old.", name, age);
    }
}

// еҗҜеҠЁжңҚеҠЎз«Ҝ
public class ProviderApplication {
    public static void main(String[] args) {
        NettyServer server = new NettyServer("127.0.0.1", 8080);
        server.registerService(HelloService.class.getName(), new HelloServiceImpl());
        
        // жіЁеҶҢеҲ°Nacos
        NacosServiceRegistry registry = new NacosServiceRegistry("127.0.0.1:8848");
        registry.registerService(HelloService.class.getName(), "127.0.0.1", 8080);
        
        server.start();
    }
}
```

### 4. и°ғз”ЁжңҚеҠЎж¶Ҳиҙ№иҖ…

```java
// rpc-consumerжЁЎеқ—
@Slf4j
public class ConsumerApplication {
    public static void main(String[] args) {
        // еҲӣе»әд»ЈзҗҶ
        NacosServiceDiscovery discovery = new NacosServiceDiscovery("127.0.0.1:8848");
        ClientProxy proxy = new ClientProxy(MessageConstants.SERIALIZER_KRYO, discovery);
        HelloService helloService = proxy.getProxy(HelloService.class);
        
        // еғҸи°ғз”Ёжң¬ең°ж–№жі•дёҖж ·и°ғз”ЁиҝңзЁӢжңҚеҠЎ
        String result = helloService.sayHello("World");
        log.info("и°ғз”Ёз»“жһңпјҡ{}", result);
    }
}
```

## рҹҺҜ йЎ№зӣ®дә®зӮ№дёҺйҡҫзӮ№

### 1. зҪ‘з»ңеұӮдјҳеҢ–
**йҡҫзӮ№**пјҡTCPзІҳеҢ…жӢҶеҢ…гҖҒNettyзәҝзЁӢжЁЎеһӢи®ҫи®Ў
**и§ЈеҶіж–№жЎҲ**пјҡ
- иҮӘе®ҡд№үеҚҸи®®еӨҙпјҲйӯ”ж•°+зүҲжң¬+зұ»еһӢ+еәҸеҲ—еҢ–+ж¶ҲжҒҜID+й•ҝеәҰпјүпјҢй…ҚеҗҲ`LengthFieldBasedFrameDecoder`зІҫеҮҶеҲҮеҢ…
- IOзәҝзЁӢдёҺдёҡеҠЎзәҝзЁӢеҲҶзҰ»пјҢйҒҝе…ҚеҸҚе°„и°ғз”Ёйҳ»еЎһEventLoop

### 2. жңҚеҠЎеҸ‘зҺ°жҖ§иғҪдјҳеҢ–
**йҡҫзӮ№**пјҡжҜҸж¬Ўи°ғз”ЁйғҪжҹҘиҜўNacosеҜјиҮҙжҖ§иғҪз“¶йўҲ
**и§ЈеҶіж–№жЎҲ**пјҡ
- жң¬ең°зј“еӯҳжңҚеҠЎеҲ—иЎЁпјҲ30з§’иҝҮжңҹпјү
- еҹәдәҺNacosзӣ‘еҗ¬жңәеҲ¶е®һж—¶жӣҙж–°зј“еӯҳ
- жіЁеҶҢдёӯеҝғж•…йҡңж—¶е…Ғи®ёдҪҝз”ЁиҝҮжңҹзј“еӯҳ

### 3. еҸҜжү©еұ•жҖ§и®ҫи®Ў
**йҡҫзӮ№**пјҡеҰӮдҪ•ж”ҜжҢҒеӨҡз§ҚеәҸеҲ—еҢ–ж–№ејҸгҖҒиҙҹиҪҪеқҮиЎЎзӯ–з•Ҙзҡ„еҠЁжҖҒеҲҮжҚў
**и§ЈеҶіж–№жЎҲ**пјҡ
- е®һзҺ°SPIжҸ’д»¶еҢ–жһ¶жһ„
- ж–°еўһе®һзҺ°зұ»еҸӘйңҖеңЁ`META-INF/services`й…ҚзҪ®еҚіеҸҜз”ҹж•Ҳ
